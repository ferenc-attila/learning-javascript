What is "this" keyword?

  • A globális kódban a global object, ami a böngészőben a window
  • Egy objektum metódusában (közvetlenül) az objektum maga
  • Függvényen belül, strict mód nélkül a gélobal object (böngészőben a window)
  • Függvényen belül, strict módban undefined
  • Egy eventnél az elem maga, amin az esemény bekövetkezett
  • bind(), call(), apply() metódusoknál bármi, magam be tudom állítani, mi legyen a this
  • konstruktor függvénynél az objektumpéldány, amit létrehozunk a segítségével
  • Az arrow function a parent scope-ból örökli a this-t (nme bind-olja a this-t)
