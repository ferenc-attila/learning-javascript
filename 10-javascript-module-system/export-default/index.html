Export default

Export default fájlonként csak egy darab lehet

Általában objektumot, osztályt szoktunk exportálni így

Nem kell object destructuring importáláskor, hiszen csak egy értéket importálhatunk így

Nem javasolt, de megadható az export default a változó létrehozásánál is. Ekkor változó deklaráció sem szükséges, egyszerűen

export default {
firstName: 'John',
lastName: 'Doe'
};
