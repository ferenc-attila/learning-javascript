Memory life cycle

Memory life cyle

  • 1: Memóriaterület lefoglalása (Allocate)
  • 2: Memóriaterület irása/olvasása (Use)
  • 3: Memóriaterület felszabadítása (Release - Garbage collection)

Változó létrehozása, értékadás

  • 1: Egyedi azonosító létrehozása
  • 2: Memóriaterület lefoglalása
  • Érték tárolása a memóriában

Adattípusok

Primitívek

  • number
  • string
  • boolean
  • undefined
  • null
  • symbol
  • bigint

let sampleNumber = 42;

let sampleNumber = 84;

A primitív típusú változó tehát immutable!

let newSampleNumber = sampleNumber;

const sampleNumber = 42;
sampleNumber += 1;

Objektumok

Minden más, ami nem primitív

  • Array
  • Map
  • Set
  • Date
  • Math
  • Function
  • ...

const sampleArray = [1,2,3,4,5];

const sampleArray = [1,2,3,4,5];
const sampleArrayCopy = sampleArray;

Ha bármelyik változó módusul, a másik is módosul

let sampleArray = [1,2,3,4,5];
let sampleArrayCopy = sampleArray;
sampleArrayCopy.push(6);
console.log(sampleArray);
console.log(sampleArrayCopy);
sampleArrayCopy = 10;
console.log(sampleArrayCopy);

Függvények esetében érték szerinti paraméterátadás történik. Objektumok esetében azonban az érték egy referencia! (Ahogyan Java esetén is)

Data type

  • number: 8 byte (IEEE 754)
  • string: 2 byte/character (UTF-16)
  • boolean: 4 byte
